# India Auto Competitor Intelligence Dashboard

A Streamlit dashboard that monitors real-time media signals for India's top 10 automotive competitors via Google News RSS.

## Run & Operate

- **Streamlit app** — workflow: `India Auto Dashboard` (port 5000)
- Start/restart: use the "India Auto Dashboard" workflow in Replit
- Data refreshes automatically every hour; click Refresh in the sidebar for immediate update

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- API: Express 5
- DB: PostgreSQL + Drizzle ORM
- Validation: Zod (`zod/v4`), `drizzle-zod`
- API codegen: Orval (from OpenAPI spec)
- Build: esbuild (CJS bundle)
